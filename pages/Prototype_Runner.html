Prototype Runner

This Project was created in during my second year at Edge Hill University while completing a module called 'Games Engines' (CIS2150). The main goal of this module was develop an understanding of of the role that game engine play in development of 3D video games and to be able to create an small game that makes use of some of the features available in the Unity game engine.

This module allowed me to develop a great understanding of C# as it was my first time programming using this language, as up to this point I had only been using Python and Java at University. Additionally, I was able to get a good understanding of how games engines such as Unity work as I hadn't used Unity before either.

Character Controller

During this project I implemented a Rigidbody based character controller. This character controller included advanced movement mechanics such as Sprinting, Wallrunning, Crouching and Sliding.

Additionally, the character controller was able to interact with Jump Pads and Boost Pads that where played though out the levels in the game.

Combat

Weapons placed in levels can be picked up by the player. While there is only 1 weapon type in the game it would be easy to implement more via polymorphism.

Enemies

The enemies use unity NavMesh for their pathfinding. They have 3 states Attacking, Chasing and Idle. During the Idle state the enemy can either stand in one spot or patrol between points of the map. During the phase state the enemy will run after the player and transition into the Attacking state when they can see the player or transition into the idle state if they have been chasing them for too long. During the Attacking state the enemy will shoot at the player.

Some levels in the game require a group of enemies to be defeated in order to progress.

UI and High Scores

This game has a fully functioning main menu witch includes a settings menu and a level select screen. Additionally when playing the game their is a timer that shows the current time since the start of the level and their is a pause menu that can be accessed by clicking escape. Finally, their is a level complete menu that will appear when the player finishes a level.

The high score of each level is tracked and saved onto the players PC. This high score is over written when the player completes a level faster then their previous high score.

Levels

There a 5 main levels in the game. These levels where created using ProBuilder and use all the mechanics that where discussed previously in on this page.
